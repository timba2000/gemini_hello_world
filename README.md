# Gemini CLI Hello World

Welcome to your first repository integrated with the Gemini CLI!

This project demonstrates a beautifully styled landing page with interactive elements (particles, smooth transitions) utilizing:
- **HTML5**
- **Vanilla CSS3** (Custom HSL theme, glassmorphism, floating spheres)
- **Vanilla JavaScript** (Particle explosion effects)
- **GEMINI.md** (Best practices configuration)

## Getting Started
1. Run Gemini CLI inside this directory:
   ```bash
   gemini
   ```
2. Open `index.html` in your browser to view the interactive hello world page.
